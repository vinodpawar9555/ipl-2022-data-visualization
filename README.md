# IPL 2022 Data Visualization Project

This project analyzes IPL 2022 cricket data using Python and data visualization libraries.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Top run scorers
- Team performance
- Match win trends
- Toss impact analysis
- Visualization plots

## Dataset
IPL 2022 dataset from Kaggle.
